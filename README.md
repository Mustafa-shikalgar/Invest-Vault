# 📖 About

Invest Vault is a modern web application designed to help users manage their investments efficiently. It provides an intuitive dashboard for tracking assets, monitoring portfolio performance, and organizing financial information securely.

Whether you're a beginner or an experienced investor, Invest Vault simplifies investment management with a clean interface and powerful features.

# 🚀 Installation

## Clone Repository

```bash
git clone https://github.com/Mustafa-shikalgar/Invest-Vault.git
```

Go to project directory

```bash
cd Invest-Vault
```

Install dependencies

```bash
npm install
```

Start development server

```bash
npm run dev
```

Open browser

```
http://localhost:5173
```

---

# ⚙ Environment Variables

Create a `.env` file in the root directory.

```env
VITE_SUPABASE_URL=your_supabase_url

VITE_SUPABASE_ANON_KEY=your_supabase_anon_key
```
